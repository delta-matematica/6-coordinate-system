Main Idea

Plotting Points

An ordered pair (x, y) tells where to place a point. Move right using the x-coordinate, then move up using the y-coordinate.

Examples
Plot (2, 3), (5, 1), and (7, 4).

Main Idea

Identifying Points

To identify a point, read the x-coordinate first, then the y-coordinate. The answer is written as an ordered pair.

Example
The point shown is (4, 3).

Main Idea

Graph of Figures

Figures on the coordinate plane are made by connecting points. Lines have endpoints, while triangles and rectangles have vertices.

Examples
Line: (1,2) to (6,2)
Rectangle vertices can be read from the corners.
